Detailed Description

template<std::size_t N, std::size_t Levels = 2>
class cudaq::qarray< N, Levels >

A qarray is an owning, compile-time sized container for qudits. The semantics of the qarray follows that of a std::array for qudits. It is templated on the number of qudits contained and the number of levels for the held qudits.
