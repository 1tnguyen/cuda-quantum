Detailed Description

The ExecutionManager provides a base class describing a concrete sub-system for allocating qudits and executing quantum instructions on those qudits. This type is templated on the concrete qudit type (qubit, qmode, etc). It exposes an API for getting an available qudit id, returning that id, setting and resetting the current execution context, and applying specific quantum instructions.
